## 👥 Adding More Users

As an admin, you can add more users:

1. Navigate to **Settings** → **Users**
2. Click **"Add User"**
3. Fill in details:
   - Name
   - Email
   - Password
   - Role (admin/viewer)
4. New user receives their credentials

**User Roles:**
- **Admin:** Full access (create, edit, delete)
- **Viewer:** Read-only access
